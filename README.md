# Odizeus Art Portfolio Website

Static portfolio website for visual artist Ulises Vargas.

## Purpose

This site showcases original artwork, including paintings, digital art, drawings, wall murals, videos, and available artwork for sale.

## Main Pages

- `index.html` - Homepage
- `paintings.html` - Paintings gallery
- `digital_art.html` - Digital art gallery
- `drawings.html` - Drawings gallery
- `wall_murals.html` - Wall murals gallery
- `videos.html` - Video gallery
- `sales.html` - Artwork currently for sale
- `about.html` - Artist biography
- `contact.html` - Contact form

## Notes

- The site is hosted with GitHub Pages.
- Gallery pages use static HTML, CSS, JavaScript, and Bootstrap.
- A backup branch was created before the cleanup work: `backup-before-cleanup-2026-05-30`.

## Maintenance Checklist

- Keep artwork image paths case-sensitive and consistent.
- Use descriptive alt text for new images.
- Add `loading="lazy"` to new gallery thumbnails.
- Avoid large framework migrations unless tested on a separate branch first.
