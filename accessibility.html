Accessibility

Plain-language notice

Last updated: June 19, 2026

Accessibility goal

Vargas Arts wants this website to be accessible, usable, and welcoming for visitors who want to view artwork, learn about Ulises Vargas, or contact the artist.

Ongoing efforts

The site is maintained with accessibility in mind where practical. Current efforts include readable contrast, descriptive alt text for artwork images, keyboard-friendly navigation where practical, responsive layouts for different screen sizes, and semantic page structure for important information pages.

Because this is an artist portfolio with image-heavy galleries and embedded media, some areas may continue to be improved over time.

Accessibility barriers

If you experience an accessibility barrier while using Vargas Arts, please contact the artist and describe the page, issue, device or browser, and any assistive technology involved if you are comfortable sharing that information.

Accessibility feedback can be sent through the contact page.
